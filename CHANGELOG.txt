## version 0.2.0 // 2024-01-30
### Added
- Using Time::HiRes module to better handle proper game delay on modern systems.

### Fixed
- minor bug fixes probably introduced with differences in software of the last 20 years.
    Things like:
    - 'aplay' for playing system wav files now rather than 'play' -- later we will try to use Audio::MPG123
    - output of aplay now sent to /dev/null
